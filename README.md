# Node.js, Express.js, MySQL API

This is a backend api for the React.js-pc-client frontend.

## Technologies used:

- Node.js
- Express.js
- MySQL
- Bcrypt
- jsonwebtoken
- Morgan
- Postman
- Sequal Ace
- Cors
- Heroku
- ClearDB
- OAuth2
- Nodemailer
